# Denial Type → Appeal Argument Patterns

## Medical Necessity (CO-50, CO-55, CO-56, CO-57)
**Strategy:** Show the service met the payer's own coverage criteria.
- Tie diagnosis → service → clinical indication
- Document conservative/prior treatment tried and failed (if policy requires step therapy)
- Cite the payer's medical policy, LCD/NCD, or named clinical guideline

**Attach:** chart notes, imaging/labs, prior treatment history, ordering physician's rationale, the referenced payer policy.

## Prior Authorization (CO-197, CO-198)
**Strategy:** Auth existed (provide the number) OR service was emergent/urgent (retrospective exception applies).

**Attach:** auth reference number, ED/urgent documentation, timeline, plan auth-requirement list.

## Timely Filing (CO-29)
**Strategy:** Prove timely submission, or that delay was payer-caused.

**Attach:** clearinghouse acceptance reports with dates, original submission records, COB correspondence.

> ⚠️ Always surface the appeal deadline — timely-filing appeals are themselves deadline-sensitive.

## Coding / Bundling (CO-97, CO-4, CO-16)
**Strategy:** Services were distinct/separately identifiable; support modifier with documentation.

**Attach:** operative/procedure notes, coding guideline reference, corrected claim.

## Non-Covered (CO-96, PR-204)
**Strategy:** Argue misclassification or medical-necessity exception. If genuinely excluded, **advise the reviewer** rather than forcing a weak appeal.

**Attach:** plan benefit language, evidence of coverage, medical-necessity documentation if an exception path exists.

## Documentation (CO-226, CO-16)
**Strategy:** Supply exactly what was requested, with a cover letter mapping each item to the attached document.

**Attach:** the specific records requested.
