# Dispute Letter Template

Populate every `[bracketed]` field from the input. Leave brackets for anything not provided and add to the reviewer checklist.

```
[Provider / Facility Letterhead]
[Provider Name] · [NPI / Tax ID]
[Address] · [Phone] · [Fax]

[Date]

[Payer Name]
Attn: Provider Disputes / Payment Integrity
[Payer Disputes Address]

RE: Payment Dispute — Underpayment on Claim [Claim Number]

Member Name:         [Patient Name]
Member ID:           [Member ID]
Claim Number:        [Claim Number]
Date(s) of Service:  [DOS]
Service / Procedure: [CPT/HCPCS + description]
Billed Amount:       $[Billed]
Expected Payment:    $[Expected] (per [contract reference / exhibit / fee schedule])
Actual Payment:      $[Paid]
Variance:            $[Variance] ([Variance %]%)

Dear Payment Integrity Team,

We are writing to dispute the reimbursement on the above-referenced claim.
Per our contract ([contract name, exhibit, effective date]), the expected
payment for [CPT code + description] is $[Expected]. The remittance dated
[payment date] shows an actual payment of $[Paid], representing a variance
of $[Variance].

ROOT CAUSE OF VARIANCE
[Explain why the payment is incorrect. Map the CARC/RARC code to the specific
error — e.g., wrong fee schedule tier applied, incorrect multiple-procedure
reduction, improper downcode, COB applied in error, contract rate not updated.]

CONTRACT REFERENCE
[Cite the specific contract provision, exhibit, fee schedule line item, and
effective date that establishes the correct rate. Only cite what you can verify.]

REQUESTED ACTION
Reprocess claim [Claim Number] and remit the underpayment difference of
$[Variance]. Please direct questions to the contact below.

ENCLOSURES
[List: copy of contract exhibit, remittance showing underpayment, fee
schedule excerpt, any supporting documentation.]

Sincerely,
[Name, Title]
[Department] · [Phone] · [Email]
```

## Formatting notes
- Keep it factual and contract-focused — this is a business dispute, not a clinical argument.
- Quote the exact CARC/RARC code and payer's adjustment reason so the dispute maps directly.
- Reference the specific contract exhibit and effective date — vague "per our contract" is weak.
- One claim per letter unless batching a pattern (then reference all claim numbers).
